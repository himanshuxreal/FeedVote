# FeedVote

DevOps-based Feedback and Voting web application implementing SCM with Git and CI/CD using GitHub Actions.



FeedVote is a simple web application that allows users to submit feedback and vote on ideas. The main purpose of this project is to demonstrate DevOps practices such as Source Code Management (SCM) using Git and GitHub, Continuous Integration (CI) using GitHub Actions, and containerization using Docker.



This project focuses on implementing a proper DevOps workflow rather than building a complex application.



Tech Stack:

Backend – Python (Flask)

Frontend – HTML, CSS

SCM – Git \& GitHub

CI/CD – GitHub Actions

Containerization – Docker

Database – MySQL (optional)



DevOps Workflow:

Developer pushes code to GitHub → Git manages version control (SCM) → GitHub Actions runs CI pipeline → Docker builds the application → Application is ready for deployment.
